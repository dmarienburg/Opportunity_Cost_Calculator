This program will use ipython to create a graph displaying the cost of leaving a position to go back to school vs
continuing in your current field.

Version Outline:
    v1.0:
        -will create the graph using user input in every field
        -graph will be created using plotly
        -time to payback and opportunity cost benefit will be displayed

    v2.0:
        -will scrape data for loans and provide options for grant based funding
        -will scrape data for salary rates of jobs related to the degree

    v3.0:
        -will scrape data for available positions in fields
        -will scrape data for annual graduates
        -will modify graphs and calculations based upon chance of employment per year
